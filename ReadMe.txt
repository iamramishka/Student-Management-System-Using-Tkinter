School Student Management System
Overview
This School Student Management System is a simple desktop application built using Python's Tkinter library and SQLite database. It's designed to help manage student records efficiently within a school setting, providing features for adding, updating, searching, and deleting student records. The system also incorporates a user-friendly GUI that makes it accessible to non-technical users.

Features
Add Student Records: Allows the user to add new student records into the system.
Update Existing Records: Users can update information of existing students.
Search for Students: Search functionality based on the student's roll number.
Delete Student Records: Users can remove student records from the database.
Clear Input Fields: Clears all the input fields in the GUI.
Persistent Data Storage: Uses an SQLite database to store student records persistently.
Prerequisites
Before you can run the application, make sure you have the following installed:

Python 3.x
Tkinter library (usually comes with Python)
tkcalendar (for the calendar date entry)
SQLite3

GUI Layout
The GUI is divided into multiple sections:

Search Section: Where you can search for students by their roll number.
Data Entry Section: Fields for entering the details of a student (roll number, name, grade, section, gender, date of birth, and attendance).
Student Record Display: Displays a table of students' records.
Control Buttons: Buttons for adding, updating, deleting, and clearing records.
